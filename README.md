# Air Writing System

An AI-based Air Writing System built using Python, OpenCV, and MediaPipe.

## Features
- Real-time hand tracking
- Smooth air writing
- White drawing canvas
- Erase mode
- Move/drag written content
- Clean split-screen UI

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## Files
- `main.py` → main application
- `hand_tracker.py` → hand detection and landmark tracking
